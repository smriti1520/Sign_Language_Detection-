How to use
  1. Click Start Camera and allow webcam access.
  2. Click Start Detection to run the hand detection loop.
  3. Snapshot saves a picture to help label data or test offline.
  4. Replace the placeholder mapping with your trained gesture model for real sign classification.
Notes

This demo visualizes hand landmarks. For production sign recognition, train a classifier (e.g., small CNN or MLP) on landmark/keypoint features or frame sequences (for dynamic signs).
